============
Installation
============

.. contents::
   :local:

You can install hop-alert-api-app via pip or from source.

To install with pip:

.. code:: bash

   pip install -U hop-alert-api-app

To install from source:

.. code:: bash

    tar -xzf hop-alert-api-app-x.y.z.tar.gz
    cd hop-alert-api-app-x.y.z
    python setup.py install
